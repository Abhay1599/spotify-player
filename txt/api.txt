- Search for Tracks: GET https://api.spotify.com/v1/search?type=track&q=<your_query>
 - Get an Artist: GET https://api.spotify.com/v1/artists/{id} 
 - Get an Album: GET https://api.spotify.com/v1/albums/{id}
 - Get an Artist's Top Tracks: GET https://api.spotify.com/v1/artists/{id}/top-tracks?country=US
 - Get Playlist Details: GET https://api.spotify.com/v1/playlists/{playlist_id}
 - Get a User's Recently Played Tracks: GET https://api.spotify.com/v1/me/player/recently-played
 - Get Recommendations Based on Seed Tracks: GET https://api.spotify.com/v1/recommendations?seed_tracks=<track_id>&limit=10
 - Create a Playlist: POST https://api.spotify.com/v1/users/{user_id}/playlists
 - Add Tracks to a Playlist: POST https://api.spotify.com/v1/playlists/{playlist_id}/tracks
 - Follow a Playlist: PUT https://api.spotify.com/v1/playlists/{playlist_id}/followers
 - Get Audio Features for a Track: GET https://api.spotify.com/v1/audio-features/{track_id}
 - Get a User's Playlists: GET https://api.spotify.com/v1/me/playlists
- Get Current User's Profile: GET https://api.spotify.com/v1/me
 - Get New Releases: GET https://api.spotify.com/v1/browse/new-releases
